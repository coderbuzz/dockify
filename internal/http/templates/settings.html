Webhook Secret

{{if .WebhookSecret}}

Use this secret in GitHub Actions to authenticate webhook deploy notifications.

GitHub Actions workflow snippet: 
- name: Notify Dockify
  env:
    DOCKIFY_WEBHOOK_SECRET: ${{"{{"}} secrets.DOCKIFY_WEBHOOK_SECRET {{"}}"}}
  run: |
    PAYLOAD='{"ref":"refs/heads/'${GITHUB_REF#refs/heads/}'","after":"'${{"{{"}} github.sha {{"}}"}}'","repository":{"clone_url":"https://github.com/'${{"{{"}} github.repository {{"}}"}}'.git"}}'
    SIG=$(echo -n "$PAYLOAD" | openssl dgst -sha256 -hmac "$DOCKIFY_WEBHOOK_SECRET" | sed 's/^.* //')
     curl -s -X POST https://dockify.example.com/api/webhook/github \
      -H "Content-Type: application/json" \
      -H "X-GitHub-Event: push" \
      -H "X-Hub-Signature-256: sha256=$SIG" \
      -d "$PAYLOAD"
{{else}}

Webhook secret is disabled. Webhooks are accepted without signature verification.

{{end}}
