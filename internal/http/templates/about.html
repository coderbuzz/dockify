Dockify

Self-hosted Docker deployment platform

What is Dockify?

Dockify is a self-hosted platform for deploying and managing Docker applications across multiple VMs. It combines a lightweight Go server, SQLite database, SSH-based worker management, and Caddy reverse proxy to provide a simple deployment workflow.

Built with Go, HTMX, SQLite, and a fully custom CSS interface — no JavaScript frameworks, no build steps.
