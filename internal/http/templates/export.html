Export all servers and apps configuration to a YAML file. SSH keys are included and encrypted with your passphrase.

⚠ All sensitive values (secrets, auth passwords, SSH keys, config files) will be encrypted. Store the passphrase safely — without it you cannot import this backup.

What's exported

  • Server: name, host, port, user, SSH key (encrypted with your passphrase)
  • Apps: name, domain, port, compose, git repo/branch, auth credentials, secrets, config files
  • Secrets are encrypted with your passphrase
  • You can edit the YAML file to remove entries before importing
